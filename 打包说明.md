# 物品位置管理器 - Android APK 打包说明

## 文件清单

| 文件 | 说明 |
|------|------|
| `item_finder.py` | 桌面版（tkinter） |
| `item_finder_kivy.py` | Kivy 移动端版 |
| `main.py` | Buildozer 打包入口（内容同 item_finder_kivy.py） |
| `buildozer.spec` | Buildozer 打包配置文件 |
| `item_finder_mobile.py` | Flet 移动端版（备用方案） |
| `.github/workflows/build-apk.yml` | GitHub Actions 自动打包配置 |

---

## 方案一：GitHub Actions 自动打包（推荐）

### 已修复的问题
- ✅ Android SDK 许可证自动接受
- ✅ build-tools 和 aidl 自动安装
- ✅ 缓存加速二次构建
- ✅ 降级到 Ubuntu 22.04 避免 libffi 编译问题
- ✅ 移除 Pillow 依赖避免 jpeg 编译问题
- ✅ 单架构构建（armeabi-v7a）减少编译时间

### 使用方法

1. **确保 GitHub 仓库包含以下文件**：
   ```
   .github/workflows/build-apk.yml
   main.py
   buildozer.spec
   ```

2. **进入 Actions 页面**，找到 "Build Android APK"

3. **点击 "Run workflow"** 手动触发

4. **等待 15-20 分钟**（首次构建下载 SDK 较慢）

5. **在 Artifacts 下载 APK**

## 方案二：本地打包（需要 Linux）

### Ubuntu/WSL

```bash
# 安装依赖
sudo apt update
sudo apt install -y build-essential libffi-dev libssl-dev libsqlite3-dev \
  pkg-config autoconf automake libtool python3-dev git ffmpeg \
  libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev \
  libportmidi-dev libswscale-dev libavformat-dev libavcodec-dev \
  zlib1g-dev libgstreamer1.0 gstreamer1.0-plugins-base \
  gstreamer1.0-plugins-good zip unzip

pip3 install kivy buildozer cython

# 进入项目目录
cd /path/to/item-finder

# 打包
buildozer android debug
```

## 方案三：Google Colab 在线打包

打开 Colab，运行以下代码：

```python
# 1. 挂载 Google Drive
from google.colab import drive
drive.mount('/content/drive')

# 2. 复制项目到 Colab
!cp -r /content/drive/MyDrive/item-finder /content/
%cd /content/item-finder

# 3. 安装依赖
!apt-get update -qq
!apt-get install -y -qq build-essential libffi-dev libssl-dev libsqlite3-dev \
  pkg-config autoconf automake libtool python3-dev git ffmpeg \
  libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev \
  libportmidi-dev libswscale-dev libavformat-dev libavcodec-dev \
  zlib1g-dev libgstreamer1.0 gstreamer1.0-plugins-base \
  gstreamer1.0-plugins-good zip unzip

!pip install -q kivy buildozer cython

# 4. 自动接受 Android SDK 许可证
import os
os.makedirs('/root/.buildozer/android/platform/android-sdk/cmdline-tools', exist_ok=True)
%cd /root/.buildozer/android/platform/android-sdk/cmdline-tools
!wget -q https://dl.google.com/android/repository/commandlinetools-linux-9477386_latest.zip
!unzip -q commandlinetools-linux-9477386_latest.zip
!mv cmdline-tools latest

# 接受许可证
!yes | /root/.buildozer/android/platform/android-sdk/cmdline-tools/latest/bin/sdkmanager --licenses --sdk_root=/root/.buildozer/android/platform/android-sdk

# 5. 打包
%cd /content/item-finder
!buildozer android debug

# 6. 复制 APK 到 Drive
!cp bin/*.apk /content/drive/MyDrive/
```
